El proceso de toma de decisiones dentro de
una empresa en materia de finanzas debe
estar siempre orientado a la rentabilidad,
liquidez y el equilibrio patrimonial. Es
fundamental comprender de qué manera
puede optimizarse el rendimiento y la
generación de valor dentro de una compañía.
Este Diplomado apunta a afianzar
conceptos clave, que permitan al alumno
aprender el manejo de herramientas de
gestión financiera, con el objetivo de tomar
decisiones que favorezcan a todo el entorno
organizacional. Está apuntado a dotar al
alumno de las competencias necesarias para
que pueda desempeñarse de manera efectiva
en el ámbito de los negocios.
Un diplomado desarrollado para lograr un
impacto positivo en las áreas de finanzas y
economía empresarial.

Plan de Estudios:
   Decisión y Análisis de Inversión    Decisión de Financiamiento y
Pago de DividendosEn este módulo se analizará la asignación de
capital de las empresas, para una correcta
inversión de los recursos de ésta, así como
se evaluará la decisión de inversión de
las organizaciones para lograr una toma
de decisiones óptima en la selección de
proyectos rentables.
A lo largo de este módulo se revisarán
las principales fuentes e instrumentos
de inversión a corto y largo plazo con los
que cuenta la empresa, para un correcto
financiamiento de sus inversiones. Finalmente
se analizará la política de dividendos y
reinversión adecuada para lograr una
estructura financiera y de capital óptima.

Descripción de los Módulos:
Diplomado en Finanzas Corporativas Avanzadas
   Estrategias de Negocios
   Valuación de Empresas
   Fusiones y Adquisiciones
Este módulo aborda los principales rubros a
analizar para establecer estrategias óptimas
para la empresa. Así como las principales
acciones de la empresa para una correcta
planeación y toma de decisiones que generen
valor.
En este módulo se aplicarán las principales
metodologías de valuación utilizadas para
una correcta estimación del valor de una
empresa o negocio y las etapas del proceso
de valuación de una empresa, para una
adecuada comprensión del valor de una
empresa.
A lo largo de este módulo se identificará el
proceso y las etapas que comprenden la
fusión o adquisición de una empresa para una
operación exitosa, además de los elementos
necesarios para una negociación e integración
exitosa.
