## Descripción
Integra la IA en la estrategia de las empresas con herramientas de AWS. 2M+ empresas mexicanas han adoptado IA; 88% reporta mejoras en productividad. Aplica servicios IA de AWS combinando visión estratégica con enfoques no-code/low-code.

## Qué hace único
- Enfoque 100% aplicable para profesionales de negocio con herramientas AWS no-code/low-code.
- Resultados medibles: cada módulo concluye con implementaciones reales que generan valor económico verificable.
- Cobertura integral del ciclo de adopción de IA: estrategia, análisis, implementación, automatización, gobernanza y ética.
- Preparación para certificación AWS Certified AI Practitioner.

## Competencias
- Integración de IA de AWS en organizaciones.
- Aplicación de IA generativa (Amazon Bedrock, SageMaker JumpStart, Amazon Q).
- Implementación de analítica predictiva con SageMaker Canvas.
- Diseño de experiencias hiperpersonalizadas (Amazon Personalize, Amazon Lex).
- Construcción de análisis con Amazon QuickSight.
- Automatización con Textract, Comprehend, Rekognition, Transcribe.
- Implementación de IA en objetivos de negocio con SageMaker Pipelines para MLOps.
- Gestión de riesgos con AWS Audit Manager, SageMaker Clarify, Bedrock Guardrails.
- Comunicación del valor de IA a diferentes audiencias organizacionales.
- Liderazgo en gestión del cambio y generación de ROI positivo.

## Plan de estudios (5 módulos × 25 horas)

### 1. IA generativa para análisis de negocio
Modelos de lenguaje de gran escala y herramientas de IA generativa. Amazon Bedrock, SageMaker JumpStart, Amazon Q, PartyRock. Prompts efectivos con ingeniería avanzada. RAG con Bedrock Knowledge Bases. Prototipado con PartyRock. Evaluación de modelos AWS.

### 2. Analítica predictiva y toma de decisiones con IA
Modelos de analítica predictiva para anticipar comportamiento de clientes y mercado. Amazon SageMaker (no-code ML), Amazon QuickSight con ML Insights. Amazon Personalize para segmentación. SageMaker Autopilot para modelos automatizados. Dashboards ejecutivos.

### 3. Automatización de procesos de negocio con IA
Amazon Textract (documentos), Comprehend (sentimientos), Rekognition (imágenes), Transcribe (conversaciones). AWS Lambda y Step Functions para flujos automatizados. Amazon Kendra para búsqueda inteligente. Marco AWS Well-Architected con ML Lens.

### 4. IA aplicada a marketing, ventas y experiencia del cliente
Ecosistemas inteligentes para predecir necesidades (Amazon Personalize). Personalización con IA generativa (Bedrock). Chatbots con Amazon Lex + Bedrock. Amazon Fraud Detector para scoring de leads. Amazon Lookout for Metrics para customer journey.

### 5. Gobernanza, ética y estrategia de implementación de IA
Marcos de gobernanza con AWS Audit Manager, Config, Artifact. Protección de datos: AWS KMS, Macie. SageMaker Clarify (algoritmos justos). Model Monitor (monitoreo continuo). Model Cards, Bedrock Guardrails, CloudTrail. Roadmaps estratégicos con SageMaker Pipelines para MLOps.
